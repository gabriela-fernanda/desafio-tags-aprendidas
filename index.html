1. Títulos: h1 até h6

(Voltar)

As tags h1 até h6 são utilizadas para definir títulos com diferentes níveis de importância. A tag h1 representa o título mais importante, enquanto h6 é usada para subtítulos de menor relevância.

Exemplo de uso:

Este é um Título Principal (h1)

Este é um Título Secundário (h2)

Este é um Título de Nível 3 (h3)

Este é um Título de Nível 4 (h4)

Este é um Título de Nível 5 (h5)
Este é um Título de Nível 6 (h6)

2. Parágrafo: p

(Voltar)

A tag p é usada para definir parágrafos de texto.

3. Destaque de Texto: mark

(Voltar)

A tag mark destaca texto como se estivesse marcado com um marcador fluorescente. Exemplo de texto destacado.

4. Texto Pequeno: small

(Voltar)

A tag small reduz o tamanho da fonte do texto. Exemplo de texto menor.

5. Texto em Itálico: i

(Voltar)

A tag i estiliza o texto em itálico. Exemplo de texto em itálico.

6. Texto Sublinhado: u

(Voltar)

A tag u sublinha o texto. Exemplo de texto sublinhado.

7. Texto em Negrito: strong

(Voltar)

A tag strong enfatiza o texto, deixando-o em negrito. Exemplo de texto em negrito.

8. Listas Ordenadas e Não Ordenadas: ol, ul e li

(Voltar)

As tags ol e ul criam listas ordenadas e não ordenadas. A tag li define cada item da lista.

Exemplo de Lista Ordenada:

  1. Primeiro item
  2. Segundo item

Exemplo de Lista Não Ordenada:

  • Primeiro item
  • Segundo item

9. Link: a

(Voltar)

A tag a cria um link para uma página ou recurso.

Target= "_blank" abre o link em outra guia. Exemplo de abertura em outra guia

Target= "_self" abre o link na mesma guia. Exemplo de abertura na mesma guia

10. Linha Horizontal: hr

(Voltar)

A tag hr insere uma linha horizontal para dividir seções de conteúdo.

11. Subscrito e Sobrescrito: sub e sup

(Voltar)

A tag sub deixa o texto subscrito, útil para fórmulas como H2O. A tag sup deixa o texto sobrescrito, como em 1o.

12. Citação em Bloco: blockquote

(Voltar)

A tag <blockquote> é usada para indicar citações em bloco:

"A alegria evita mil males e prolonga a vida." - William Shakespeare

13. Fonte: font

(Voltar)

A tag font é usada para alterar o tipo de fonte de um texto em HTML. Ela está desaprovada em HTML5. Exemplo: Texto em azul, tamanho 4 e fonte Arial.

14. Texto Riscado: del

(Voltar)

A tag del indica que o texto foi removido ou substituído, aparecendo riscado. Exemplo: Texto que foi removido ou substituído.

15. Abreviação: abbr

(Voltar)

A tag abbr define uma abreviação e pode exibir o significado ao passar o mouse sobre o texto. Exemplo: EUA.
